# PUCIT ResourceHub — Frontend

This is the React + Vite frontend for **PUCIT ResourceHub**.

For full setup instructions, contribution guidelines, and project documentation, see the [root README](../README.md) and [CONTRIBUTING.md](../CONTRIBUTING.md).

## Quick Start

```bash
npm install
cp .env.example .env   # fill in VITE_API_URL
npm run dev            # http://localhost:5173
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start the Vite dev server |
| `npm run build` | Build for production |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview the production build |
